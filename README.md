# ContentDetector


Detect hard/soft skills from resumes in Russian.

Это модуль, который из текста резюме/вакансии вычленяет провессиональные навыки (например, 1C, .NET) и "мягкие" навыки (обучаемость, коммуникабельность и т. п.).
